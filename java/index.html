Qu'est-ce que Java

Java est une le nom d’une technologie et langage d’Oracle. Cela permet de faire tourner des applications écrits dans ce langage. La plateforme Java permet peut se soustraire du système d’exploitation, ce dernier faisant la liaison. Ainsi une même application Java peut tourner sur Windows ou Linux ou Ios. Enfin, des applications Java peuvent être lancées depuis le navigateur WEB, on parle alors d’applet Java, à ne pas confondre avec JavaScript qui n’a rien à voir. C'est aussi une language tourner objet.

Qu'est-ce qu'une applet Java

Une applet est un programme Java qui s'exécute dans un logiciel de navigation supportant Java ou dans l'appletviewer du JDK.

Comment intégrer une applet

<APPLET NAME="monapplet" CODE="monapplet.class" height=400 width=700 </APPLET>

Les différent type de variable

byte : Le type byte (1 octet) peut contenir les entiers entre -128 et +127.

short : Le type short (2 octets) contient les entiers compris entre -32768 et +32767.

int : Le type int (4 octets) va de -2*109 à 2*109 (2 et 9 zéros derrière… ce qui fait déjà un joli nombre).

long : Le type long (8 octets) peut aller de −9×1018 à 9×1018 (encore plus gros…).

float : Le type float (4 octets) est utilisé pour les nombres avec une virgule flottante.

double : Le type double (8 octets) est identique àfloat, si ce n'est qu'il contient plus de chiffres derrière la virgule et qu'il n'a pas de suffixe.

char : Le type char contient un caractère stocké entre apostrophes (« ' ' »), comme ceci :

boolean : Le type boolean, lui, ne peut contenir que deux valeurs :true(vrai) oufalse(faux), sans guillemets (ces valeurs sont natives dans le langage, il les comprend directement et sait les interpréter).

String : Le typeStringpermet de gérer les chaînes de caractères, c'est-à-dire le stockage de texte (ATTENTION : String commence par une majuscule)

A l'aide qu'es ce qu'un octet

Un octet equivaut a 8 bits.

On en vien a qu-es ce qu'un bits, un bits et une valeurs qui est soit 1 soit 0. pour mieux expliquer cela voici un example:

0101 0110

dans cette chaine de 8 bits donc 1 octet ,a gauche ce situe le bit de poid lourd et a droite le bite de poid faible. le bit de poid faible vaut 1, celuis a sa gauche vaut 2 fois ça valeur, celuis a gauche de ce précedent vaut encore 2 fois la valleur de son precedent

Ensuite, on aditionne tout les valleur ou les bits sont a 1 pour obtenir un valeur en décimale

64+16+4+2 = $d86

pour 2 octet

0010 0110 0001 0111
